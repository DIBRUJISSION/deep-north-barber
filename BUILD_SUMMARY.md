## 🚀 Quick Start

### 1. Install Dependencies
```bash
cd "d:\Web development\Barber"
npm install
```

### 2. Run Development Server
```bash
npm run dev
```
Open `http://localhost:3000` in your browser.

### 3. Preview All Sections
The homepage includes:
- 🎯 Hero section with strong headline
- 🛡️ Trust bar with 6 benefits
- 👤 About/Ervin founder story
- 📋 9 services with pricing
- ⭐ Why Choose differentiation
- 💰 2 promotional offers
- 🖼️ Gallery (6-item results)
- 📍 Location, hours, booking
- 🎢 Final CTA
- 🔗 Footer with links

### 4. Customize
Edit key files:
- **Services/Pricing**: `src/components/Services.tsx`
- **Copy/Messaging**: Any component (easily editable)
- **Colors**: `tailwind.config.js` (brand color palette)
- **Contact Info**: `src/components/Location.tsx` + `Footer.tsx`
- **Images**: Replace placeholders with real photos

### 5. Deploy
Pick one:
- **Vercel** (recommended): Connect GitHub, auto-deploy
- **Netlify**: Connect GitHub, easy setup
- **Self-hosted**: Use DEPLOYMENT.md guide
